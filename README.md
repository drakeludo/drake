# 🍊 Majestic RP: Orange Collector AI (Colab Optimized)

Этот репозиторий содержит всё необходимое для обучения нейросети YOLOv8, которая находит апельсины в мини-игре на Majestic RP.

## 🚀 Быстрый старт в Google Colab (Рекомендуется)

Самый простой способ обучить модель — использовать наш автоматизированный ноутбук. Он сам установит все зависимости, настроит GPU и будет сохранять результаты на ваш Google Drive.

1. Откройте файл **`colab_auto_train.ipynb`** в этом репозитории.
2. Нажмите кнопку **"Open in Colab"**.
3. Следуйте инструкциям внутри ноутбука.

## 📂 Структура проекта

- **`colab_auto_train.ipynb`** — Главный ноутбук для обучения в облаке.
- **`train_colab.py`** — Оптимизированный скрипт обучения (авто-выбор GPU, 100 эпох, сохранение чекпоинтов).
- **`download_data.py`** — Скрипт для автоматической подготовки структуры папок датасета.
- **`yolo_dataset/`** — Папка для вашего датасета (изображения и разметка).
- **`best.pt`** — Текущие лучшие веса модели.

## 🛠 Как подготовить датасет

Так как изображения занимают много места, они не хранятся в репозитории. 
1. Соберите скриншоты апельсинов из игры.
2. Разметьте их (например, в [Roboflow](https://roboflow.com) или LabelImg).
3. Положите изображения в `yolo_dataset/images/train` и текстовые файлы разметки в `yolo_dataset/labels/train`.
4. В Colab ноутбуке есть инструкция, как загрузить ваш датасет через Google Drive.

## 📈 Параметры обучения

Мы настроили обучение так, чтобы оно было стабильным в Colab:
- **Модель:** YOLOv8n (самая быстрая и легкая).
- **Эпохи:** 100 (с ранней остановкой).
- **Размер картинки:** 640x640.
- **Автосохранение:** Результаты дублируются на ваш Google Drive в папку `YOLO_Orange_Sniper`.
